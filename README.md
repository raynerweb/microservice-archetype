# microservice-archetype
Archetype with SpringBoot, Spring JPA and Jersey. 

Steps:

- git clone git@github.com:raynerweb/microservice-archetype.git
- cd microservice-archetype
- mvn install


For update:
- git pull
- mvn install archetype:update-local-catalog
